Detailed Description

Smart Port protocol uses 8 bytes packets.

Packet format (byte): tiivvvvc

  • t: type (1 byte)
  • i: sensor ID (2 bytes)
  • v: value (4 bytes - int32)
  • c: crc

The uint64 presentation is much easier to use for data shifting.
